# Font Files

Place your Aeonik and Suisse Int'l font files in the respective directories:

## Aeonik (Primary - Headings)
Place in `/public/fonts/aeonik/`:
- `Aeonik-Regular.woff2`
- `Aeonik-Medium.woff2`
- `Aeonik-Bold.woff2`

## Suisse Int'l (Secondary - Body)
Place in `/public/fonts/suisse/`:
- `SuisseIntl-Regular.woff2`
- `SuisseIntl-Medium.woff2`

Font files should be in `.woff2` format for optimal performance.
